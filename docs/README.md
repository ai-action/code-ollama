# Code Ollama Documentation

Code Ollama is an [Ollama](https://ollama.com/) coding agent that runs in your terminal.

## Quick Start

Set up [Ollama](ollama.md), then run:

```sh
npx code-ollama
```

## Getting Started

- [Ollama](ollama.md) — install the server, choose a model, and configure the host
- [TUI](tui.md) — use modes, tools, commands, shortcuts, files, and images
- [CLI](cli.md) — run one-off prompts, diagnostics, and saved sessions

## Features

- [MCP](mcp.md) — connect external tools and resources
- [Skills](skills.md) — add reusable instructions and workflows
- [Memory](memory.md) — preserve global and project notes
- [Sessions](sessions.md) — resume, compact, and inspect conversations
- [Web Search](web-search.md) — configure SearXNG and DuckDuckGo fallback

## Reference

- [Configuration](configuration.md) — review settings, precedence, validation, and security

## Motivation

Why use Code Ollama?

- **free** (no per-token API cost)
- **private** (local inference)
- **offline** (run without internet)
- **open models** (Gemma, Qwen, DeepSeek, etc.)
